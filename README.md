# Login & Registration.
# Introduction :
This project demonstrates a responsive user Registration and Login system (SignUp & SignIn) using the MERN stack:
M: MongoDB (Database)
E: ExpressJS (Backend Framework)
R: ReactJS (Frontend Framework)
N: NodeJS (Backend Runtime)

# Installation & Getting Started 
1. Clone the Repository
git clone <repository-url>
cd <project-folder>
2. Frontend Setup
cd frontend
npm install        # Install dependencies
npm run dev        # Start development server
